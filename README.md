# My First Project

Стартовый проект для веб-приложения.

## Установка

```bash
npm install
```

## Запуск dev-сервера

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура

- `index.html` — главная страница
- `server.js` — локальный Node.js сервер
- `package.json` — конфигурация проекта
